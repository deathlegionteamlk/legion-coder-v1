# LEGION Coder v1 - Installation Guide

## Quick Start

### Windows Users
1. Download `LEGION Coder v1.exe` (172.5 MB)
2. Double-click to run - **No installation required!**
3. Get your free OpenRouter API key at https://openrouter.ai/keys
4. Paste the key in Settings → API Key
5. Start coding! 🚀

### Android Users
1. Download `app-release.apk` (4.3 MB)
2. Enable "Install from Unknown Sources" in Settings
3. Tap the APK to install
4. Configure your OpenRouter API key
5. Start coding on mobile! 📱

---

## Detailed Installation

### Windows Installation

#### System Requirements
- **OS**: Windows 10/11 (64-bit)
- **RAM**: 4 GB minimum, 8 GB recommended
- **Storage**: 200 MB free space
- **Internet**: Required for AI features

#### Step-by-Step Installation

1. **Download the EXE**
   ```
   File: LEGION Coder v1.exe
   Size: 172.5 MB
   SHA256: 1fa93c3471c11dc8128998c662c705104e4528b98901b61c8a25216acda424c5
   ```

2. **Run the Application**
   - Double-click `LEGION Coder v1.exe`
   - If Windows SmartScreen appears, click "More info" → "Run anyway"
   - The app is portable - no installation needed!

3. **Create Shortcut (Optional)**
   - Right-click the EXE → "Create shortcut"
   - Move to Desktop for easy access

4. **Configure API Key**
   - Open Settings (Ctrl+, or gear icon)
   - Go to "API Configuration"
   - Enter your OpenRouter API key
   - Select preferred model

#### Troubleshooting Windows

| Problem | Solution |
|---------|----------|
| "Windows protected your PC" | Click "More info" → "Run anyway" |
| Missing DLL errors | Install [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe) |
| App won't start | Right-click → "Run as administrator" |
| Black screen | Update graphics drivers |

### Android Installation

#### System Requirements
- **OS**: Android 8.0+ (API 26+)
- **RAM**: 2 GB minimum, 4 GB recommended
- **Storage**: 50 MB free space
- **Internet**: Required for AI features

#### Step-by-Step Installation

1. **Download the APK**
   ```
   File: app-release.apk
   Size: 4.3 MB
   SHA256: 6e4857a4fb13b04ed415524a2fd920477108e375cb69d8c6de1db882c26a8c75
   ```

2. **Enable Unknown Sources**
   
   **Android 8.0+ (Oreo and newer):**
   - Settings → Apps → Special Access → Install Unknown Apps
   - Select your browser/file manager
   - Enable "Allow from this source"
   
   **Android 7.0 and older:**
   - Settings → Security → Unknown Sources
   - Enable the toggle

3. **Install the APK**
   - Open Downloads folder or notification
   - Tap `app-release.apk`
   - Tap "Install" when prompted
   - Wait for installation to complete

4. **Launch the App**
   - Tap "Open" after installation
   - Or find "LEGION Coder" in app drawer

5. **Configure API Key**
   - Tap Settings (gear icon)
   - Enter OpenRouter API key
   - Select AI model

#### Troubleshooting Android

| Problem | Solution |
|---------|----------|
| "Install blocked" | Enable "Unknown Sources" in Settings |
| "App not installed" | Uninstall previous version first |
| "Parse error" | Download APK again, check file integrity |
| App crashes | Clear app data in Settings → Apps |
| AI not responding | Check internet connection and API key |

## API Key Setup

### Getting Your Free API Key

1. Visit https://openrouter.ai/keys
2. Create a free account
3. Generate an API key
4. Copy the key (starts with `sk-or-...`)

### Configuring in LEGION Coder

**Windows:**
1. Press `Ctrl + ,` or click gear icon
2. Paste API key in "OpenRouter API Key" field
3. Select model from dropdown
4. Click "Save"

**Android:**
1. Tap gear icon
2. Paste API key in API Key field
3. Select model
4. Tap "Save"

### Available Free Models

| Model | Description | Best For |
|-------|-------------|----------|
| `google/gemma-3-27b-it` | Google's Gemma 3 | Fast responses |
| `mistralai/mistral-7b-instruct` | Mistral 7B | Balanced quality |
| `meta-llama/llama-3.1-8b-instruct` | Llama 3.1 | Complex reasoning |

## First Use Guide

### Creating Your First Project

1. **Open LEGION Coder**
2. **Create/Open Folder**
   - Windows: File → Open Folder (Ctrl+K Ctrl+O)
   - Android: Tap folder icon → "Open Folder"
3. **Create New File**
   - Windows: Ctrl+N or File → New File
   - Android: Tap "+" button
4. **Start Coding**
   - Select language from dropdown
   - Type your code
   - Use AI chat for assistance (Ctrl+Shift+A)

### Using AI Features

1. **Open AI Chat**
   - Windows: Ctrl+Shift+A or click AI icon
   - Android: Tap AI icon

2. **Ask Questions**
   - "Explain this code"
   - "Refactor this function"
   - "Generate tests for this"

3. **Apply Suggestions**
   - Review AI response
   - Click "Apply" to insert code
   - Or copy manually

## Uninstallation

### Windows
Simply delete `LEGION Coder v1.exe` - no registry entries or files left behind.

Settings are stored in:
- `%APPDATA%/legion-coder/` (Windows)

### Android
1. Settings → Apps → LEGION Coder
2. Tap "Uninstall"
3. Confirm
